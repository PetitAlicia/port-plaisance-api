Description de l'API

Cette application de Gestion des Réservations de Catway permet de gérer les utilisateurs (créer ou supprimer un utilisateur), les catways (créer, modifier, supprimer ou afficher les détails d'un catway), et les réservations (créer, supprimer ou afficher les détails d'une réservation). Elle utilise le standard REST pour les routes et repose sur une base de données MongoDB pour stocker les informations.

Fonctionnalités

Gestion des utilisateurs - Authentifier, créer et supprimer des utilisateurs.

Gestion des catways - Créer, modifier et supprimer des catways, ainsi qu'afficher la liste des catways et les détails d'un catway précis.

Gestion des réservations - Créer et supprimer des réservations, ainsi qu'afficher la liste des réservations et les détails d'une réservation précise.

Méthodes et routes

Utilisateurs

POST /api/auth/signup - Authentifier un utilisateur
POST /api/auth/login - Créer un utilisateur
DELETE /api/auth/delete - Supprimer un utilisateur

Catways

POST /api/catways - Créer un catway
PUT /api/catways/:id - Modifier un catway
DELETE /api/catways/:id - Supprimer un catway
GET /api/catways/:id - Afficher les détails d'un catway

Réservations

POST /api/catways/:id/reservations - Créer une réservation
DELETE /api/catways/:id/reservations/:id - Supprimer une réservation
GET /api/catways/:id/reservations/:id - Afficher les détails d'une réservation

Pages

index.html - La page d'accueil, permettant de se connecter
dashboard.html - Le tableau de bord, donnant accès aux différentes fonctionnalités
catways.html - La liste des catways, ainsi que les détails de chaque catway
reservations.html - La liste des réservations, ainsi que les détails de chaque réservation
documentation.html - La documentation de l'API

Exemples

Exemples de réponse de l'API

Afficher les détails d'un catway

Requête :

GET /api/catways/605c72d73e6e241b6c9e9e1e

Réponse (JSON) :

{
"_id": "605c72d73e6e241b6c9e9e1e",
"catwayNumber": 1,
"type": "long",
"catwayState": "disponible"
}

Créer une réservation

Requête :

POST /api/catways/:id/reservations

Corps de la requête (JSON) :
{
"catwayNumber": "1",
"clientName": "John Doe",
"boatName": "Le Neptune",
"checkIn": "2024-10-20T14:00:00Z",
"checkOut": "2024-10-25T10:00:00Z"
}
Réponse (JSON) :
{
"_id": "605c72d73e6e241b6c9e9e20",
"catwayNumber": "605c72d73e6e241b6c9e9e1e",
"clientName": "John Doe",
"boatName": "Le Neptune",
"checkIn": "2024-10-20T14:00:00Z",
"checkOut": "2024-10-25T10:00:00Z"
}

Glossaire

API - Interface de programmation d'applications permettant la communication entre différentes applications.

Catway - Petit appontement pour amarrer un bateau.

MongoDB - Base de données utilisée pour stocker les données de l'application.

Réservation - Engagement pour l'utilisation d'un catway à une certaine période.

REST - Architecture pour la création de services web.
